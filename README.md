# IT Plus Team Project

# How to download this project:
Type the following in your terminal:

```
git clone https://github.com/Jocundity/it-plus-team-project.git
```

# How to contribute to this project:
Before starting any new work, run the pull command in your terminal to make sure you are working from the latest version:

```
git pull origin main
```

When you're done editing your files, commit your changes to the repo by entering these commands into your terminal:

```
git add .
git commit -m "Brief description of your changes"
git push -u origin
```
